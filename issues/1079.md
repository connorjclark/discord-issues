```cpp
npc script doFiring
{
    void run()
    {
        while(true)
        {
            if(this->Tile >= this->OriginalTile + 80)
            {
                while(this->Tile >= this->OriginalTile + 80)
                    Waitframe();
                this->Attack();
            }
            Waitframe();
        }
    }
}```
That should work for you
When it hits the end of the 'firing' animation, it will shoot it's set weapon
